# snapped-blogs
### This is a small learning project.

<ins>Key things in the project:</ins>
- Frontend using EJS.
- Backend using Pyhton Flask.
- Database using MongoDB Atlas.
- Docker integration.
- Kubernetes deployments,services and ingress.

To run this project on local : 
- Utilizing docker to run as containers.
- - Run ``` docker compose up ```
  - This will start 2 docker containers :
  - - frontend-0
    - backend-0
  - The ports for both frontend and backend are defined in the docker-compose file:
  - - frontend - 3000
    - backend - 8000
  - I have also defined the MongoDB Atlas URL for my project (please do not use 😰)
  

### I have utilised a kubernetes cluster on Google Cloud Engine to host the project.
